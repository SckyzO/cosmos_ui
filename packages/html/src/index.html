🎨 CSS Design Tokens
Include in any HTML project — zero framework dependency:
<link rel="stylesheet" href="cosmos-tokens.css" >

<div class="cosmos-root dark" >
  /* Uses --color-brand-500, --color-bg-base, etc. */
</div>

--color-brand-500
→ #465fff (indigo default)
--color-bg-base
→ #0a0a0a (dark bg)
--color-accent
→ current accent color
.cosmos-root
→ app shell class
